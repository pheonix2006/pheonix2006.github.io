# pheonix2006 个人主页设计方案

> 创建日期：2026-03-10

## 1. 项目概述

**站点名称：** pheonix2006 的个人主页
**域名：** pheonix2006.github.io
**部署平台：** GitHub Pages

**核心理念：**
- 以 Markdown 文件为内容源，写项目记录就像写文档一样简单
- 温暖文艺的视觉风格：柔和配色、圆角卡片、舒适的阅读体验
- 长期友好的结构：内容与代码分离，未来迁移或扩展都不受影响

## 2. 技术选型

| 项目 | 选择 | 理由 |
|------|------|------|
| 框架 | Astro | 零JS默认、快速、Markdown友好 |
| 样式 | Tailwind CSS | 灵活定制、原子化 |
| 语言 | TypeScript | 类型安全、长期维护 |
| 基础主题 | Astro Persona | 结构完整、设计简洁 |

## 3. 站点结构

```
pheonix2006.github.io/
├── 首页 (/)          → 个人简介 + 最近动态
├── 项目 (/projects)  → 做过的项目展示
├── 博客 (/blog)      → 技术笔记/生活记录
├── 关于 (/about)     → 更详细的自我介绍
└── 联系 (/contact)   → 社交链接/邮箱
```

## 4. 页面设计

### 首页
- 顶部：个人头像 + 一句简短的自我介绍
- 中间：最近项目（3-4个卡片）+ 最新文章（3-5条列表）
- 底部：社交图标链接（GitHub、邮箱等）

### 项目页
- 卡片网格布局，每个卡片包含：项目封面图、名称、简述、技术标签
- 点击进入详情页：完整介绍、截图、源码/演示链接
- 支持分类筛选（如：前端、后端、工具等）

### 博客页
- 文章列表 + 日期 + 阅读时长
- 支持标签分类（如：技术、随笔、学习笔记）
- 文章详情页：舒适的阅读排版、代码高亮

## 5. 视觉风格

- **配色：** 暖白背景 `#faf8f5` + 柔和橙/棕作为强调色
- **字体：** 中文用思源宋体/霞鹜文楷，英文用衬线体
- **元素：** 圆角卡片 + 轻阴影，整体感觉温暖舒适

## 6. 文件结构

```
src/
├── content/
│   ├── blog/           ← 博客文章放这里
│   └── projects/       ← 项目记录放这里
├── pages/              ← 页面模板（基本不用改）
└── components/         ← 组件（按需定制）

public/
└── images/             ← 图片素材放这里
```

## 7. 内容管理流程

### 写一篇新博客
在 `src/content/blog/` 下新建 `.md` 文件：

```markdown
---
title: 文章标题
date: 2026-03-10
tags: [标签1, 标签2]
image: ./cover.png
---

正文内容...
```

### 添加一个新项目
在 `src/content/projects/` 下新建 `.md` 文件：

```markdown
---
title: 项目名称
date: 2026-03-10
tags: [Vue, TypeScript]
image: ./cover.png
github: https://github.com/xxx
demo: https://xxx.com
---

## 项目背景
项目介绍...

## 技术栈
- Vue 3
- TypeScript

```
